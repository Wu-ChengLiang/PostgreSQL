# 部署脚本说明

## 🚀 主要部署脚本

### `start-services.sh`
启动本地服务的脚本

### `rebuild-frontend.sh`
重新构建前端应用

## 🔧 修复脚本

这些脚本用于修复部署过程中遇到的各种问题：

- 数据库连接问题
- 前端构建错误
- API路由配置
- 服务启动问题

## 📝 使用说明

1. 确保所有脚本有执行权限：`chmod +x *.sh`
2. 根据需要运行相应的脚本
3. 检查日志输出确认操作成功
